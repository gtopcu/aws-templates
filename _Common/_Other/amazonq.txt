Inline Chat: Mac + I

1. /dev
Ask for help implementing a new feature. Q analyzes your project context and suggests an approach with code examples

2. @workspace
Get answers about your entire codebase, not just the open file. Try "@workspace how is authentication handled?"

3. /transform
Automatically upgrade Java code - ie. move Java 8 projects to Java 17
