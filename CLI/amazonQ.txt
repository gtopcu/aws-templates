Code:
Code complete: Option + C
Inline Chat: Command + i

Console:
Tab: Display all options
Left arrow to accept autocompletion

q chat  @history to pass your shell history
        @git to pass information about your current git repository
        @env to pass your shell environment

/clean     - Clear current dialog
/doc       - Generate documentation
/dev       - Ask for help implementing a new feature
/test      - Generate unit tests
/review    - Get recommendations
/transform - Automatically upgrade Java code - ie. move Java 8 projects to Java 17

@workspace - Get answers about your entire codebase, not just the open file. 
Example -> "@workspace how is authentication handled?"
