q chat  @history to pass your shell history
        @git to pass information about your current git repository
        @env to pass your shell environment

/dev - Ask for help implementing a new feature. Q analyzes your project context and suggests an approach with code examples

@workspace - Get answers about your entire codebase, not just the open file. Try "@workspace how is authentication handled?"

/transform Automatically upgrade Java code - ie. move Java 8 projects to Java 17
